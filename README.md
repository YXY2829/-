# Enterprise-Registration-Data-of-Chinese-Mainland

中国大陆31个省份1978年至2019年一千多万工商企业注册信息，包含企业名称、注册地址、统一社会信用代码、地区、注册日期、经营范围、法人代表、注册资金、企业类型。

数据以CSV文件形式存储，格式如下：

企业名称,注册地址,统一社会信用代码,地区,注册日期,经营范围,法人代表,注册资金,企业类型,所在省份
